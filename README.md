# Plataforma Rosa — Teologia para Mulheres

Landing page moderna e responsiva para o curso online de teologia "Plataforma Rosa".

## 🎯 Características

- Hero responsivo com layout otimizado para mobile e desktop
- Seção de professoras com cards de imagens padronizadas
- Grade curricular com scroll automático
- Seção de investimento com preços e bônus
- FAQ interativa
- Formulário de inscrição
- Design moderno com cores quentes e acessíveis
- 100% responsivo (mobile-first approach)

## 📁 Estrutura

```
pagina rosa/
├── plataforma-rosa.html  # HTML principal
├── styles.css                          # Estilos CSS
├── scripts.js                          # JavaScript (animações e interatividade)
├── imagens/                            # Fotos das professoras
│   ├── prof-a.png
│   ├── prof-t.png
│   └── ...
└── vercel.json                         # Configuração Vercel
```

## 🚀 Deploy

### GitHub
- Repositório: https://github.com/viniciuscanavieira/Doksa-pagina1

### Vercel
1. Conecte seu repositório GitHub ao Vercel
2. Selecione este projeto
3. Clique em "Deploy"

O site estará disponível em: `https://doksa-pagina1.vercel.app/`

## 📸 Imagens das Professoras

Para adicionar as fotos das professoras, substitua os caminhos em `plataforma-rosa.html`:
- `images/prof-a.png` → Miss. Aderian Henrique
- `images/prof-t.png` → Profa. Tereza Silva
- `images/prof-r.png` → Profa. Rejane Sousa
- E assim por diante...

As imagens devem ter:
- Dimensão: Quadradas (1:1 aspect ratio)
- Tamanho recomendado: 500x500px ou superior
- Formato: PNG ou JPG
- Pasta: `/images/`

## 🎨 Customização

### Cores (variables em styles.css)
- `--rosa`: cor principal
- `--ameixa`: tons secundários
- `--creme`: background principal

### Fontes
- Títulos: Poppins (bold)
- Corpo: Poppins (regular)
- Monospace: DM Mono
